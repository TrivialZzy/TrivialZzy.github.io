内置数据结构

list

函数和方法 描述 注
append() 列表末尾添加一个元素
pop(idx=-1) 返回指定位置元素并删除 从0开始, 默认删除最后一个
sort(reverse=True, key=...) 排序 默认从小到大, key可以自定义排序规则
count(val) 返回val出现的次数
index(val) 返回val所在的位置 保证val存在, 从0开始
insert(idx, val) 在下标idx处添加元素val 下标从0开始
remove(val) 删除第一个值为val的项 保证val存在
extend(lst) 将可迭代元素添加到列表末尾 lst为可迭代元素
copy() 创建新列表, 值是列表的副本
clear() 清空列表

str

函数和方法 描述 注
split(s) 在s处拆分字符串, 并返回列表 原字符串不变, 返回结果
strip(s) 去除字符串两端的s字符, 返回改变后的字符串 原字符串不变, 返回结果
find(s) 返回s在字符串中出现的位置 从0开始, 找不到返回-1
replace(s1, s2) 替换s1为s2, 并返回替换后的结果 原字符串不变, 返回结果
count(s) 返回s在字符串中出现的次数
lower() 将字符串变为小写 原字符串不变, 返回结果
islower() 若所有字符都是小写, 返回True
upper() 将字符串变为大写 原字符串不变, 返回结果
isupper() 若所有字符都是大写, 返回True
isalpha() 若所有字符都是字母, 返回True
isdigit() 若所有字符都是数字, 返回True
isalnum() 若所有字符都是字母或数字, 返回True
title() 每个单词首字符转化为大写 原字符串不变, 返回结果

注: str无法更改
