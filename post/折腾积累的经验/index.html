安装或配置各种东西时积累的经验, 以macOS为例

macOS删除系统自带ABC输入法

  1. 安装可以修改.plist文件的编辑器
    安装Xcode或者其他可以修改.plist文件的编辑器
  2. 关闭系统的SIP
    以M系列处理器的mac为例
    • 在关机的情况下长按开机键进入恢复模式
    • 选择选项, 点击继续, 登录管理员账号
    • 点击实用工具, 打开终端
    • 运行如下命令后重启系统 
      1

      csrutil disable
    注: 关闭SIP的步骤类似, 只是输入的命令改为 
    1

    csrutil enable
  3. 修改文件删除ABC输入法
    • 重启后切换到ABC输入法, 打开终端, 执行如下命令 
      1

      sudo open ~/Library/Preferences/com.apple.HIToolbox.plist
    • 依次点开Root - AppleEnabledInputSources, 找到其中的KeyboardLayout Name为ABC的项, 删掉整个项, 之后保存文件
  4. 重启系统, 删除完成
    注: 若想找回ABC输入法, 直接添加键盘即可

安装colorls

  1. 安装ruby 
    1

    brew install ruby
  2. 给gem换源 
    1

    gem sources --add https://gems.ruby-china.com/ --remove https://rubygems.org/
  3. 下载本地文件
    去这个网站下载本地文件
  4. 本地安装 
    1

    gem install -l colorls-1.4.6.gem
    注: 去到文件所在的文件夹运行, 名称以当时下载的文件为准
  5. 修改shell的配置文件
    以zsh为例, 去.zshrc添加如下语句 
    1

    PATH=$PATH:$(ruby -e 'puts Gem.bindir')
  6. 重启终端, 完成安装
    注: 可以在.zshrc中添加如下的alias语句, 方便使用 
    1
2
3

    alias ls="colorls"
alias ll="colorls -l"
alias la="colorls -a"

Markdwon中使用中文表格|对不齐的解决方案

下载并使用Sarasa Mono SC字体, 参考这篇文章

具体步骤如下

  1. 下载Sarasa Mono SC字体
    去https://github.com/be5invis/Sarasa-Gothic下载sarasa-gothic-ttf
  2. 安装对应字体
    安装Sarasa Mono SC字体, Regular/Light/Bold/Italic即可
  3. 设置编辑器
    以VSCode为例, 编辑Font Family加入Sarasa Mono SC
