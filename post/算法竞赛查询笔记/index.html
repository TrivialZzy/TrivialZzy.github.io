数据范围与时间复杂度

数据范围 时间复杂度
$n\le30$ $O(2^n)$
$n\le100$ $O(n^3)$
$n\le1000$ $O(n^2)$, $O(n^2\log{n})$
$n<10000$ $O(n\sqrt{n})$
$n\le10^5$ $O(n\log{n})$
$n\le10^6$ $O(n)$, $O(n\log{n})$
$n\le10^7$ $O(n)$
$n\le10^9$ $O(\sqrt{n})$
$n\le10^{18}$ $O(\log{n})$

    *t = 72;
    (*t)++;
}
int f2(int& t) {
    t = 72;
    t++;
}

int x = 0, y = 0;
f1(&x), f2(y); // 两个函数的作用是相同的
printf("%d %d\n", x, y); // 结果: 73 73

struct Vector {
    int x, y;

    Vector() : x(0), y(0) {}
    Vector(int _x, int _y) : x(_x), y(_y) {}

    // 重载 <
    bool operator < (const Vector& other) {
        if (x > other.x) {
            return true; // x降序
        } else {
            return y < other.y; // y升序
        }
    }
    // 重载 +
    Vector operator + (const Vector&);
};

Vector Vector::operator + (const Vector& other) {
    return Vector(x+other.x, y+other.y);
}

元素访问

函数 作用
at() 访问指定元素, 进行越界检查, 速度慢
operator[] 访问指定元素, 不进行越界检查, 速度快

容量相关

函数 作用
empty() 检查容器是否为空
size() 返回容纳的元素数, 永远等于size_max()

增改操作

函数 作用
fill() 以指定值填充容器
swap() 交换内容, 前提: 具有相同的长度和类型; 时间复杂度: $O(size)$
