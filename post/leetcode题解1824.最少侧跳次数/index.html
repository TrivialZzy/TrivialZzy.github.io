1824. 最少侧跳次数

题解

分析

记 $dp[i][j]$ 是到达第 $i$ 个点第 $j$ 条跑道的最小侧跳次数, 记 $(i, j)$ 为第 $i$ 个点第 $j$ 条跑道的位置, 其中 $0\le i\le n, j = 1, 2, 3$.

由题意知, $dp[0][2] = 0, dp[0][1] = dp[0][3] = 1$, 记 $(i, j)$ 位置为障碍时 $dp[i][j] = INF$.

下面对于 $1\le i\le n$ 的情况进行考虑.

固定 $j$:

  • 若 $(i, j)$ 为障碍, 则 $dp[i][j] = INF$;

  • 若 $(i, j)$ 不为障碍:

    • 若 $dp[i-1][j] = INF$, 则无法通过 $(i-1, j)$ 到达 $(i, j)$, 此时先记 $dp[i][j] = INF$;
    • 若 $dp[i-1][j] \neq INF$ , 则可以通过 $(i-1, j)$ 到达 $(i, j)$, 此时先记 $dp[i][j] = dp[i-1][j]$.

    此时可以看出: $dp[i][j] = dp[i-1][j]$.

经过上述讨论, $dp[i][j]$ 的值只取决于第 $i$ 个点剩下两条跑道侧跳到第 $j$ 条跑道的情况了, 并且从其他两条跑道侧跳过来之前, 一定不在第 $i$ 个点的三条跑道上. 记 $k$ 为其他两条跑道的任意一条, 则此时 $dp[i][j]$ 的情况只取决于 $(i, k)$ 的状态和 $dp[i-1][k]$ 的情况, 而这与上面讨论得到的 $dp[i][k]$ 有关.

我们进行接下来的讨论, 不妨只讨论 $j = 1$ 的情况, 对于 $j = 2, 3$ 的情况有类似的结果, 不在此赘述.

此时容易看出

$$ dp[i][1] = \min(dp[i][1], dp[i][2]+1, dp[i][3]+1) = \min(dp[i][1], \min(dp[i][2], dp[i][3])+1), $$

从而最终的结果为

$$ \min(dp[n][1], dp[n][2], dp[n][3]). $$

至此, 题目分析完毕.

优化

  • 我们在建立数组 $dp[n+1][4]$ 时, $dp[i][0]$ 始终没有用到, 故我们可以记三条跑道的编号分别为 $0, 1, 2$ , 此时可以节约空间.

  • 我们发现考虑 $dp[i][j]$ $(1\le i\le n, j = 0, 1, 2)$ 的值时只与 $(i, j)$ 的状态和 $dp[i-1][j]$ 的值有关, 故我们可以用数组 arr[3] 去存 $dp[i][j]$ $(0\le i\le n, j = 0, 1, 2)$ 的值, 滚动更新, 可以节约空间.

class Solution {
public:
    int minSideJumps(vector<int>& ob) {
        const int INF = 0x3f3f3f3f;
        int N = ob.size(); // 此时 N = n+1
        array<int, 3> arr{1, 0, 1};
        for (int i=1; i<N; i++) {
            for (int j=0; j<3; j++) {
                if (ob[i]==j+1) { // 此时表示 (i, j) 的位置为障碍; 当不为障碍时, 由分析和优化知, 值不会变
                    arr[j] = INF;
                }
            }
            // (j, j+1, j+2) 刚好是 3的剩余类 的代表元素
            for (int j=0; j<3; j++) {
                if (ob[i]!=j+1) { // 只有当 (i, j) 的位置不为障碍时才能由其他跑道侧跳到达
                    arr[j] = min(arr[j], min(arr[(j+1)%3], arr[(j+2)%3])+1);
                }
            }
        }
        return min(arr[0], min(arr[1], arr[2]));
    }
};
